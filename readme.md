# Data Analyst Job Scraper Portfolio Project

## Overview
This project scrapes job listings from ejobs.ro, cleans the data, and visualizes trends. Built with Python, BeautifulSoup, Pandas, and Matplotlib.

## Files
- `data_analyst_job_scraper.ipynb`: The main Jupyter notebook.
- `data_analyst_job_scraper.html`: HTML export for easy viewing.
- `ejobs_data_analyst_cleaned.csv`: Scraped and cleaned data.

## How to Run
1. Install dependencies: `pip install -r requirements.txt` (create a `requirements.txt` with your packages).
2. Open the notebook and run the cells.
